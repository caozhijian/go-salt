## Getting started

```
# clone the project
git clone https://github.com/jcops/go-salt.git

# enter the project directory
cd vue-front

# install dependency
npm install

# develop
npm run dev
```

This will automatically open [http://localhost:9527](http://localhost:9527/)

## Build

```
# build for test environment
npm run build:stage

# build for production environment
npm run build:prod
```
